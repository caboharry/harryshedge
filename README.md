# Harry's Hedge

## Project Description
Harry's Hedge is a web application designed to provide users with an intuitive platform for managing and tracking their investments in a clear and efficient manner.

## Features
- User-friendly dashboard for tracking investments
- Portfolio management tools
- Real-time market data and analytics
- Notifications for significant market changes
- Secure user account management

## Getting Started
To get started with Harry's Hedge, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/caboharry/harryshedge.git
   ```

2. Navigate to the project directory:
   ```bash
   cd harryshedge
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```

4. Run the application:
   ```bash
   npm start
   ```

## Tech Stack
- Frontend: React.js
- Backend: Node.js with Express
- Database: MongoDB
- Authentication: JSON Web Tokens (JWT)
